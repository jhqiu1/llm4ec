A curated collection of open-source tools, code implementations, and benchmark suites for **LLM-driven algorithm design and evolutionary computation**.

> ⚠️ **This page is continuously updated.** New resources are being added regularly.

---

## Platforms

Comprehensive frameworks for LLM-based automatic algorithm design.

| Platform | Description | Link |
|----------|-------------|------|
| **LLM4AD** | Open-source Python platform for Automatic Algorithm Design with 100+ tasks and 10+ methods, developed by CityU OPTIMA Group. | [GitHub](https://github.com/Optima-CityU/LLM4AD) |
| **BLADE** | Benchmark suite for LLM-driven Automated Design and Evolution of iterative optimisation heuristics, developed at LIACS, Leiden University. | [GitHub](https://github.com/XAI-liacs/BLADE) |
| **EASE** | Effortless Algorithmic Solution Evolution — a flexible framework leveraging LLMs to generate solutions (algorithms, text, images) based on user-defined parameters. | [GitHub](https://github.com/TBU-AILab/EASE) |

## Code Repositories

Open-source implementations of key methods, organized by research paradigm.

### LLM as Designer (LLMaD)

| Method | Venue | Description | Link |
|--------|-------|-------------|------|
| **EoH** | ICML 2024 (Oral) | Evolution of Heuristics — LLM-driven automatic heuristic design via evolutionary search | [GitHub](https://github.com/FeiLiu36/EoH) |
| **FunSearch** | Nature 2023 | Mathematical discovery via program search with LLMs and evolutionary algorithms | [GitHub](https://github.com/google-deepmind/funsearch) |
| **Eureka** | ICLR 2024 | Human-level reward design for reinforcement learning via coding LLMs | [GitHub](https://github.com/eureka-research/Eureka) |
| **ReEvo** | NeurIPS 2024 | LLMs as hyper-heuristics with reflective evolution for combinatorial optimization | [GitHub](https://github.com/ai4co/LLM-as-HH) |
| **LLaMEA** | IEEE TEVC 2024 | LLM Evolutionary Algorithm for automatically generating novel metaheuristics | [GitHub](https://github.com/nikivanstein/LLaMEA) |
| **ADAS** | ICLR 2025 | Automated Design of Agentic Systems via meta-agent search | [GitHub](https://github.com/ShengranHu/ADAS) |
| **ELM** | GECCO 2022 | Evolution Through Large Models — pioneering LLM-driven code evolution | [GitHub](https://github.com/CarperAI/OpenELM) |
